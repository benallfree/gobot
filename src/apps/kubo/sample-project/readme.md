# `kubo` sample project

Features:

- Demo `"bin"` script
- Shows how to run `kubo` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.34/docs/readme.md).
- Shows how to add [gobot-kubo](https://www.npmjs.com/package/gobot-kubo) as a version-locked dependency.
