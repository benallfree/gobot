# `pulumi` sample project

Features:

- Demo `"bin"` script
- Shows how to run `pulumi` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.20/docs/readme.md).
- Shows how to add [gobot-pulumi](https://www.npmjs.com/package/gobot-pulumi) as a version-locked dependency.
