# `sftpgo` sample project

Features:

- Demo `"bin"` script
- Shows how to run `sftpgo` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.37/docs/readme.md).
- Shows how to add [gobot-sftpgo](https://www.npmjs.com/package/gobot-sftpgo) as a version-locked dependency.
