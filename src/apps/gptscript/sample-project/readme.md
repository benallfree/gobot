# `gptscript` sample project

Features:

- Demo `"bin"` script
- Shows how to run `gptscript` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.23/docs/readme.md).
- Shows how to add [gobot-gptscript](https://www.npmjs.com/package/gobot-gptscript) as a version-locked dependency.
