# `pocketbase` sample project

Features:

- Demo `"bin"` script
- Shows how to run `pocketbase` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.31/docs/readme.md).
- Shows how to add [gobot-pocketbase](https://www.npmjs.com/package/gobot-pocketbase) as a version-locked dependency.
