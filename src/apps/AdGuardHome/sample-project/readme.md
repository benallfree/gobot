# `AdGuardHome` sample project

Features:

- Demo `"bin"` script
- Shows how to run `AdGuardHome` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.17/docs/readme.md).
- Shows how to add [gobot-adguardhome](https://www.npmjs.com/package/gobot-adguardhome) as a version-locked dependency.
