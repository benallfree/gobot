# `rclone` sample project

Features:

- Demo `"bin"` script
- Shows how to run `rclone` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.26/docs/readme.md).
- Shows how to add [gobot-rclone](https://www.npmjs.com/package/gobot-rclone) as a version-locked dependency.
