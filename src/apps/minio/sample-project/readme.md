# `minio` sample project

Features:

- Demo `"bin"` script
- Shows how to run `minio` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.16/docs/readme.md).
- Shows how to add [gobot-minio](https://www.npmjs.com/package/gobot-minio) as a version-locked dependency.
