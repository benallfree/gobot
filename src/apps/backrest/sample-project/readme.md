# `backrest` sample project

Features:

- Demo `"bin"` script
- Shows how to run `backrest` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.28/docs/readme.md).
- Shows how to add [gobot-backrest](https://www.npmjs.com/package/gobot-backrest) as a version-locked dependency.
