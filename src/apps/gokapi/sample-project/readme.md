# `gokapi` sample project

Features:

- Demo `"bin"` script
- Shows how to run `gokapi` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.35/docs/readme.md).
- Shows how to add [gobot-gokapi](https://www.npmjs.com/package/gobot-gokapi) as a version-locked dependency.
