# `caddy` sample project

Features:

- Demo `"bin"` script
- Shows how to run `caddy` programmatically using the [Gobot API](https://github.com/benallfree/gobot/tree/v1.0.0-alpha.33/docs/readme.md).
- Shows how to add [gobot-caddy](https://www.npmjs.com/package/gobot-caddy) as a version-locked dependency.
