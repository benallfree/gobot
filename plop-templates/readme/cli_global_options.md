**Global Options**

| Option                  | Default       | Discussion                                   |
| ----------------------- | ------------- | -------------------------------------------- |
| `--g-help`              | Show help     | Display help and options for Gobot           |
| `--g-v[vv]`             |               | Adjust output verbosity                      |
| `--g-cache-path <path>` | host specific | Use the specified directory for cache files. |
